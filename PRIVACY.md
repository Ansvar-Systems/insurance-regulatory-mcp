# Privacy Policy

## Data Processing

This MCP server is a **read-only regulatory reference tool**. It:

- Processes no personal data
- Stores no user queries or session data
- Contains no analytics, telemetry, or tracking
- Makes no outbound network requests during normal operation (all data is served from a local SQLite database)
- Requires no user accounts or authentication for tool calls

## Data Stored

The server's SQLite database contains only publicly available regulatory publications: IAIS Insurance Core Principles, IAIS Application Papers, ComFrame materials, NAIC Model Laws, and Lloyd's Market Requirements. No personal data is stored.

## Premium Tier

Access control for premium features is handled by the Ansvar platform layer, not by this MCP server directly. This server does not process billing or subscription data.
